# Git Commit & Push Workflow

// turbo-all

- `/cp`: Khi nhận được lệnh này, hãy thực hiện các bước sau:
    1. Chạy `git add .` để stage tất cả các thay đổi.
    2. Tạo message commit theo chuẩn Conventional Commits dựa trên các thay đổi đã thực hiện. Message commit nên bằng tiếng Việt và tuân thủ các quy tắc trong .cursorrules.
    3. Thực hiện commit với message đã tạo.
    4. Chạy `git push` để đẩy code lên repository.
